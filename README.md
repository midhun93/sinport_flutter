# Flutter UI Implementation Test – Sinport Style

This project demonstrates a Flutter UI implementation inspired by the Sinport Airport mobile UI design.

## Features
- Clean Flutter widget structure
- Reusable components
- Responsive layout
- Gradient header
- Horizontal airport cards
- Bottom navigation

## Project Structure

lib/
  main.dart
  screens/
    home_screen.dart
  widgets/
    header_widget.dart
    service_item_widget.dart
    airport_card_widget.dart
    bottom_nav_widget.dart
  utils/
    colors.dart

## Run

flutter pub get
flutter run
